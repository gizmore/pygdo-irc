# pygdo-irc
[PyGDO](https://github.com/gizmore/pygdo) module to add an IRC connector. Basic IRC functionality.

## Installation

```
cd pygdo
./gdo_adm.sh provide -y irc
pygdo add_server name01 IRC tcps://irc.wechall.net:6697
```
